IDM241 Microinteraction - Beta Description

https://www.youtube.com/

Triggers

Create Button:

  • Hover on
  • Hover off
  • Click without dropdown
  • Click with dropdown

Create Button Dropdown Options:

  • Hover on
  • Hover off

Rules

Create Button:

  • Hover on
    • Button color gets darker. Ease out transition in 150ms
    • Cursor changes to finger
  • Hover off
    • Button color gets darker. Ease out transition for 150ms
    • Cursor changes back to default pointer
  • Click without dropdown
    • Button color gets more dark
    • Dropdown menu opens. Quick Smart Animate transition for 600ms
    • Button's plus icon spins once
  • Click with dropdown
    • Button color returns back to a lighter red color
    • Dropdown menu closes. Quick Smart Animate transition for 600ms
    • Button's plus icon spins once

Create Button Dropdown Options:

  • Hover on
    • Image slightly scales up
    • Cursor changes to finger
  • Hover off
    • Image slightly scales down
    • Cursor changes back to pointer
